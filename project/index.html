よくあるご質問

参加前の疑問にお答えします

Q 歌が苦手な子でも参加できますか？ +

もちろんです！ジョジョ・アコスタ氏は世界中の初心者から経験者まで指導してきた実績があります。このワークショップは「歌うことの楽しさ」を第一に考えており、技術よりも自信と表現力の向上に重点を置いています。

Q 英語が分からなくても大丈夫ですか？ +

全く問題ありません！日本語でのサポートスタッフも常駐します。

Q 保護者の見学は可能ですか？ +

はい、問題ないです！

Q 持ち物はありますか？ +

特別な持ち物は必要ありません。

Q 参加費用はかかりますか？ +

完全無料です！世界的なボイストレーナーによる特別ワークショップを、多くのお子様に体験していただきたいという想いから、参加費は一切いただいておりません。ただし、定員20名限定となっておりますので、お早めにお申し込みください。

Q 兄弟姉妹での参加は可能ですか？ +

もちろん可能です！対象年齢（小学生〜中学生）内であれば、兄弟姉妹での参加を歓迎いたします。ただし、お一人ずつお申し込みをお願いいたします。

Q 申し込み後のキャンセルは可能ですか？ +

キャンセルは可能です。体調不良やご都合が悪くなった場合は、前日までにメール（globalbunny77@gmail.com）でご連絡ください。無料のイベントですので、キャンセル料は発生しません。ただし、他の参加希望者のためにも、お早めのご連絡をお願いいたします。

Q アクセス方法を教えてください +

UDCK（柏の葉アーバンデザインセンター）へのアクセス：

  • 🚃 つくばエクスプレス「柏の葉キャンパス駅」より徒歩1分
  • 🚗 駐車場：近隣の駐車場をご利用ください
  • 🚌 バス：柏駅東口より阪東バス「柏の葉キャンパス駅西口」下車
  • 📍 住所：千葉県柏市柏の葉5-1-5

詳細な地図は下部のGoogleマップをご確認ください。
